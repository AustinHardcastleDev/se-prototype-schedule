### Learnings:
- **React Router v6 Nested Routes:** Use `<Outlet />` in layout component to render child routes. The layout component wraps all routes and provides persistent UI (sidebar, top bar).
- **Responsive Sidebar Pattern:** Use Tailwind `hidden md:flex` on sidebar and `md:hidden` on mobile top bar for clean responsive switching at breakpoint.
- **Slide-out Drawer Animation:** Use `transform translate-x-full` for off-screen positioning and `translate-x-0` with transition for smooth slide-in animation.
- **PropTypes in React 18:** ESLint requires PropTypes validation for all component props. Import from 'prop-types' package.
- **date-fns for Date Formatting:** `format(date, 'MMMM yyyy')` provides clean month/year display in mobile top bar.
- **NavLink Active States:** React Router's `NavLink` component provides `isActive` in render prop for conditional styling. Use orange bg for active, hover state for inactive.
- **Fixed Positioning for Mobile Top Bar:** Mobile top bar uses `fixed top-0 left-0 right-0` with `z-50` to stay at top during scroll. Main content needs `pt-14` to account for fixed header height.
